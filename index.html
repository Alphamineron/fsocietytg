    Anaconda

    It is an environment manager tool for python. What do I mean by that?
    Well, you'll do many projects with python. Each project has its own requirements, a specific set of libraries to be used in that project. Conda helps us create a separate environment for each project.
           For eg - one of your projects needs python 3.7 but other one needs version 3.6, so if you make separate virtual conda environments for each of your projects you can choose exactly which libraries and versions to install and they will be installed only for that particular environment.
    Download the Python3.x version from Anaconda's Official Website...

    • Refer to this cheat sheet if you're looking for a conda command. Otherwise, do a thorough google search, that always works out.

    $ conda create -n first_project - Create a new environment named first_project

    $ conda activate first_project - Activate the environment

    Now, any python program you run through the terminal will use the libraries installed in the environment (first_project) and any modules you install from the web will also be installed in that environment and not in your Operating System's Python Installment, which as you might guess, you should avoid messing with.

    Universal Function Approximators

    Ultimately, A Neural Network is a networked combination of linear functions (y = ax +b) combined with non-linear activation functions (takes inputs and returns -1 to 1, or 0-1, etc.).

    By combining linear with non-linear, and lots of them, and providing a training process to refine the linear functions settings (via weights and bias), it allows a neural network to approximate nearly any type of function, and thus any problem and solution. Hence, the reference to neural networks as ‘Universal function approximators’.
